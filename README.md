# 🚀 CryptoVision: AI-Powered Cryptocurrency Price Prediction Platform

An intelligent **Machine Learning powered web application** that predicts cryptocurrency prices using the **XGBoost algorithm** and real-time financial data.

This system provides a **data-driven platform for analyzing cryptocurrency trends, forecasting prices, and visualizing market insights** through an interactive web interface.

The project was developed as a **Final Year Project for the Bachelor of Science in Computer Science at COMSATS University Islamabad**.

---

# 🧠 Project Overview

Cryptocurrency markets are highly volatile and unpredictable. This project introduces a **machine learning powered prediction system** that helps users analyze trends and forecast future prices.

The platform collects historical data from **Yahoo Finance**, processes it using **Machine Learning models**, and predicts future cryptocurrency prices.

Users can:

- Track cryptocurrencies
- Analyze price movements
- Generate price predictions
- View visualized analytics
- Manage accounts through a secure authentication system

Administrators can manage coins, users, and system data through a dedicated **admin dashboard**.

# 🤖 Machine Learning Model

The prediction system is powered by **XGBoost (Extreme Gradient Boosting)**.

### Why XGBoost?

- High prediction accuracy
- Handles large datasets efficiently
- Fast training performance
- Works well with financial time-series data

### Prediction Workflow

1️⃣ Collect cryptocurrency historical data  
2️⃣ Preprocess and clean dataset  
3️⃣ Train XGBoost model  
4️⃣ Generate predictions for next-day price  
5️⃣ Display predictions through dashboard
